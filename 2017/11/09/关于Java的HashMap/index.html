关于HashMap的其他细节

每一扩容都是2的n次方：

首先为什么每一次都是2的n次方这是由于hash函数导致的，假设一个值是1001，那么另一个值可以是1101,1011,1111，这三种情况做且运算的话结果都是1001，发生了三次碰撞，那么假设固定值是2的n次方的话，减一之后是1111，折三个数与其做且运算的话都不会相同，所以会较小碰撞次数。
